MLX AudioGen lets you generate professional-quality music and audio from text prompts directly on your Apple Silicon Mac. Using the MLX framework, all inference runs on your local Metal GPU — no API keys, no usage limits, no data leaving your machine. It supports MusicGen (autoregressive), Stable Audio Open (diffusion), and Demucs v4 stem separation, with a full stack of interfaces: CLI, HTTP server, React web UI, VST3/AU plugin, and Max for Live.

## Get started in four steps

<Steps>
  <Step title="Install">
    Install via pip with your desired feature set:
    ```bash
    pip install mlx-audiogen[server]
    ```
  </Step>
  <Step title="Generate audio from the CLI">
    Run your first generation — models auto-download on first use:
    ```bash
    mlx-audiogen --model musicgen --prompt "upbeat jazz piano" --seconds 10
    ```
  </Step>
  <Step title="Launch the web UI">
    Open a DAW-style browser interface for interactive generation:
    ```bash
    mlx-audiogen-app
    ```
  </Step>
  <Step title="Connect your DAW">
    Use the VST3/AU plugin or Max for Live device to generate audio directly inside Ableton Live or Logic Pro.
  </Step>
</Steps>
